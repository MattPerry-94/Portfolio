Sujet N°2 : Framework Vue.js

Sujet : Comment Vue.js a-t-il évolué pour répondre aux nouvelles exigences du web moderne ?

Pourquoi ce sujet ?

J’ai choisi d’étudier l’évolution de Vue.js car ce framework JavaScript est largement utilisé pour le développement d’interfaces web modernes. Avec l’arrivée de Vue 3, le framework a introduit des améliorations majeures en matière de performance, de modularité et de compatibilité avec TypeScript. Cette évolution montre comment les frameworks frontend s’adaptent aux nouvelles exigences du web moderne : performance, maintenabilité et optimisation SEO.

Points analysés :

  • Passage de Vue 2 à Vue 3
  • Introduction de la Composition API
  • Améliorations de performance (Virtual DOM)
  • Meilleure intégration avec TypeScript
  • Architecture SPA vs SSR
  • Écosystème moderne (Router, Pinia, Vite)
