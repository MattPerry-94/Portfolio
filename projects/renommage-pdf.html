Renommage PDF

Script Python - Automatisation

Description du projet

Script Python d'automatisation conçu pour traiter et renommer massivement des fichiers PDF en fonction de leur contenu ou de métadonnées spécifiques. Le script assure ensuite l'archivage automatique de ces documents vers une bibliothèque SharePoint et notifie les équipes concernées par email avec un rapport d'exécution.

Fonctionnalités principales

  • Extraction de données depuis les PDF
  • Renommage automatique selon conventions
  • Upload automatique sur SharePoint via API
  • Envoi de rapport d'exécution par email

Technologies utilisées

Python
